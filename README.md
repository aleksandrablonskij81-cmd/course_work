# Курсовая работа: Мониторинг самолётов

## Описание
Программа получает данные о самолётах в воздушном пространстве выбранной страны через API OpenSky.

## Функциональность
- Получение координат страны через Nominatim API
- Получение списка самолётов через OpenSky API
- Сохранение данных в JSON-файл
- Вывод топ N самолётов по высоте
- Фильтрация по стране регистрации

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Тесты
```bash
pytest tests/ --cov=src --cov-report=term
```
